## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/yourusername/eShop.git
```

### Open the Project

```bash
cd eShop
```

### Run Locally

Use:

* Live Server (VS Code)
* XAMPP
* Any local web server

Then open:

```text
http://localhost:5500
```
